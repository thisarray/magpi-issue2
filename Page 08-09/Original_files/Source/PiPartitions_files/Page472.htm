Resize Partitions

In order to maximise the use of the SD-Card, it is recommended the main partition of the card is enlarged to fill up the remaining “unallocated” space on the card.

 

To do this, you will need a partition management program, GParted is recommended (unfortunately most Windows partition software can not manipulate the Linux formatted partitions on the SD-Card).  However, you can use a LiveCD or VM (Virtual Machine), such as Puppy/RacyPy, Debian - which were covered in The MagPi Issue 1.  There is even a specific GParted LiveCD, available with instructions on how to use it:

http://gparted.sourceforge.net/livecd.php

 

If not already installed on your Linux system, gparted can also be obtained with the following command in a terminal window:

sudo apt-get install gparted

 

Running GParted

 

We now need to start the GParted utility you installed previously.  Now you may think you can now start GParted by finding it in the start menu, but no.  This utility requires admin rights, so at your terminal prompt type:

 

sudo gparted

 

And press enter.  Type your password (if prompted) and press enter.  Wait until the GParted utility loads.  Now click the dropdown box in the upper right hand corner, and you should see your SD card that we identified in the previous steps.  Click that.

 

Disclaimer: We are going to be altering disk partitions, so you MUST make sure you've selected the right disk (card in our case).  If your screenshot differs vastly from that above, stop and ask yourself some stern questions before continuing.  I'm not going to be responsible if you end up mucking about with your installed system.  Remember always, you are running this as a super user.  Please be careful.
